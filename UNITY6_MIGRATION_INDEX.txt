CRITICAL CHANGES AT A GLANCE
================================================================================

TOTAL CHANGES: 3 categories, ~20 code changes, 1 complete rewrite

1. VELOCITY PROPERTY RENAME (6 changes)
   rb.velocity → rb.linearVelocity
   Files: PlayerController.cs (2), EnemyBase.cs (4)
   Priority: 🔴 CRITICAL - Game will have physics bugs without this

2. UI TEXT MIGRATION (5+ field changes)
   Text → TextMeshProUGUI
   Files: UIManager.cs, GameSetupEditor.cs
   Priority: 🟠 HIGH - UI will not compile/work without this

3. TEXTMESH DEPRECATION (1 class rewrite)
   TextMesh → TextMeshPro 3D
   Files: TextPopup.cs
   Priority: 🟠 HIGH - Text popups won't display without this

WHAT IF SOMETHING GOES WRONG?
================================================================================

Compilation Errors:
  → Check MIGRATION_SUMMARY.txt "SPECIFIC CODE CHANGES" section
  → Look up the exact file and line number
  → Compare with CODE_CHANGES_REQUIRED.md for context

Runtime Issues (Game Behaves Strangely):
  → Check for missed "rb.velocity" references
  → Verify UI text is properly assigned
  → See TESTING CHECKLIST for what should work

Text Not Displaying:
  → Verify TextMeshPro changes in TextPopup.cs
  → Check GameSetupEditor.cs CreateUIText() rewrite
  → Confirm using TMPro; is added

Physics Bugs (Player falls through floor, weird jumping):
  → This is definitely a missed velocity change
  → Search for all rb.velocity (should be none)
  → Replace with rb.linearVelocity

UI Not Working:
  → Check if Text → TextMeshProUGUI changes applied
  → Verify using TMPro; added to files
  → Confirm no old UnityEngine.UI.Text references remain
